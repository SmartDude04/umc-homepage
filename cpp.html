C++

C++ - A programing language famous for its use in computer operating systems and core components of applications

C++ is one language in the C family of programming languages. The most popular languages in this family are C, C++, and C#. C and C++ were both created before 2000, and C# was created by microsoft in 2000. C++ is known for having a very steep learning curve, and it's object oriented style makes it a tough language for many to grasp. Before C++, C was created to design operating systems and critical software to make computers work. It's still used today for that purpose. After C++, C# was created. C# is the easiest of the languages to learn, and its use in app development engines like Unity makes it widely popular.

Language Use Difficulty (Lower is Easier)
C Low-level applications, IoT, OS Kernals, Embedded Software
C++ Operating Systems, Games Web Browsers, Compilers
C# Mobile Apps, Games, Web Apps, Cloud Services, Software
