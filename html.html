HTML

HTML - A language famous for creating the world wide web as we now know it

HTML, along with CSS, create the framework for the world wide web. Instead of being a programming language, HTML is a markup language, which means that it puts elements of a website where they should be. CSS, its companion, is a styling language. Its job is to style the HTML on the website to make it look nice.
