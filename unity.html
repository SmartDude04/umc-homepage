Unity

Unity - A program famous for making 2D and 3D games

Now yes, Unity is not a programming language. But its use of C# and C++ to run games makes it close enough to put on this list. Plus, it's super cool. Unity is a program that allows anyone to make a game without going through the hastle of making their own game engine. Its use of both blocks to make the layout of the game as well as code to run the game makes it very appealing.
